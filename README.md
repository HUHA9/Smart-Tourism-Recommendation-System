# Smart Tourism Recommendation System

This project is my Final Year Project.

It is a smart tourism recommendation system designed to provide personalized travel routes and destination suggestions based on user preferences.
---
## Key Features
- Personalized travel destination recommendations
- Travel route planning based on selected cities
- Interactive map and attraction display
- Favorite (liked) destinations management
- Multi-page tourism information system
---
## Technologies Used
- HTML5
- CSS3
- JavaScript
- Map API (for route and attraction display)
---
## System Pages
- `index.html` – Homepage
- `city.html` – City selection and attraction information
- `travel-plan.html` – Travel route planning
- `liked.html` – Favorite destinations
- `history.html` – Travel history
- `mytrip.html` – User trip management
